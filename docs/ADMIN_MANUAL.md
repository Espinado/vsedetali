# Документация панелей управления

Руководства разделены по аудитории:

| Файл | Для кого | Панель |
|------|----------|--------|
| **[PLATFORM_MANUAL.md](./PLATFORM_MANUAL.md)** | Сотрудники магазина-площадки | `/admin` |
| **[SELLER_MANUAL.md](./SELLER_MANUAL.md)** | Сотрудники компании-продавца | `/seller` |

Учётные записи **не взаимозаменяемы**: вход покупателя с витрины и вход в одну из панелей — это разные типы доступа.
