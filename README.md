# vsedetalki.ru — интернет-магазин автозапчастей

Рабочий проект. Стек: Laravel 12 + Livewire. Перспектива: B2C → B2B → Marketplace.

## Документация

- [Архитектура](docs/ARCHITECTURE.md) — модули, сущности, связи, роли, roadmap.
- [Структура БД](docs/DATABASE_STRUCTURE.md) — таблицы, поля, индексы, FK, порядок миграций.

## Открытие как отдельный проект

Откройте в Cursor/VS Code файл **`vsedetali.code-workspace`** (или папку `C:\laragon\www\vsedetali`) — это отдельный воркспейс, не связанный с fleet.
