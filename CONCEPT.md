# vsedetalki — концепция интернет-магазина автозапчастей

## Идея проекта

**vsedetalki** — интернет-магазин автозапчастей на Laravel 12 + Livewire 3: быстрый каталог с фильтрами, корзина, личный кабинет и отдельная админ-панель для управления товарами и заказами.

---

## Стек технологий

| Компонент | Решение |
|-----------|--------|
| Backend | Laravel 12 |
| Фронт (каталог, корзина, ЛК) | Livewire 3 + Alpine.js (из коробки) |
| Стили | Tailwind CSS |
| Админ-панель | **Filament 3** (админ на Livewire, быстрое CRUD) или кастом на Livewire |
| Авторизация | Laravel Breeze (логин/регистрация) + роли (user / admin) |
| БД | MySQL (через Laragon) |

---

## Роли и зоны сайта

1. **Публичная часть** — каталог, карточка товара, корзина, оформление заказа, страницы «О нас», «Доставка», «Контакты».
2. **Личный кабинет (user)** — после входа: мои заказы, профиль, смена пароля, история заказов.
3. **Админ-панель (admin)** — только для роли admin: товары, категории, заказы, пользователи, при необходимости — слайдеры, страницы.

## Структура данных (основные сущности)

### Каталог

- **Category** — дерево категорий (кузов, двигатель, ходовая, электроника и т.д.). Поля: name, slug, parent_id, sort_order, is_active.
- **Product** — запчасть. Поля: name, slug, sku (артикул), category_id, price, old_price, quantity, description, short_description, is_active, meta_title, meta_description. Связь с брендами/моделями авто — по желанию на втором этапе (марка/модель/год).
- **ProductImage** — несколько фото на товар (порядок, is_main).
- **Brand** (опционально) — производитель запчасти или марка авто.

### Корзина и заказы

- **Cart** — корзина (session или user_id для авторизованных). Храним: session_id или user_id.
- **CartItem** — позиция: cart_id, product_id, quantity, price (снимок на момент добавления).
- **Order** — заказ. Поля: user_id (nullable для гостя), status (new, confirmed, paid, shipped, completed, cancelled), total, delivery_address, delivery_type, phone, email, comment, created_at.
- **OrderItem** — позиция заказа: order_id, product_id, product_name, price, quantity.

### Пользователи и авторизация

- **User** — стандартная модель Laravel + поле **role** (user | admin).
- Регистрация и вход — через Breeze (или кастомные Livewire-компоненты с теми же правилами).

## Основные сценарии (флоу)

### 1. Каталог

- Главная: блоки «Популярные», «Новинки», категории, поиск.
- Каталог: список категорий → список товаров в категории с фильтрами (цена, наличие, бренд и т.д.).
- Поиск по названию/артикулу (по мере роста — полнотекстовый или Scout).
- Карточка товара: галерея, цена, количество, кнопка «В корзину», описание, артикул, доставка.

### 2. Корзина

- Живая корзина на Livewire: добавление/удаление/изменение количества без перезагрузки.
- Мини-корзина в шапке (иконка + количество).
- Страница корзины: список позиций, итог, кнопка «Оформить заказ».

### 3. Оформление заказа

- Шаг 1: контакты (имя, телефон, email). Для авторизованных — подтягиваем из профиля.
- Шаг 2: доставка (способ: самовывоз/курьер/почта, адрес).
- Шаг 3: подтверждение заказа, создание Order и OrderItem, очистка корзины, редирект в ЛК или на страницу «Заказ оформлен».

### 4. Личный кабинет

- Профиль: имя, email, телефон, смена пароля.
- Мои заказы: список заказов, детали заказа (статус, состав, сумма).

### 5. Админ-панель

- Дашборд: кол-во заказов за день/неделю, низкий остаток товаров.
- Категории: дерево, CRUD, сортировка.
- Товары: список с фильтрами, создание/редактирование (включая галерею и артикул).
- Заказы: список, смена статуса, просмотр деталей.
- Пользователи: список, смена роли (user/admin).

## Предлагаемая структура папок и ключевые Livewire-компоненты

```
app/
├── Livewire/
│   ├── Catalog/
│   │   ├── ProductGrid.php          # Сетка товаров + фильтры
│   │   ├── ProductCard.php          # Карточка в списке
│   │   └── ProductShow.php          # Страница товара
│   ├── Cart/
│   │   ├── CartIcon.php             # Иконка корзины в шапке
│   │   ├── CartPage.php             # Страница корзины
│   │   └── AddToCartButton.php      # Кнопка «В корзину»
│   ├── Checkout/
│   │   └── CheckoutWizard.php      # Пошаговое оформление заказа
│   ├── Account/
│   │   ├── Profile.php              # Редактирование профиля
│   │   └── OrderList.php            # Список заказов пользователя
│   └── ...
├── Models/
│   ├── Category.php
│   ├── Product.php
│   ├── ProductImage.php
│   ├── Cart.php, CartItem.php
│   ├── Order.php, OrderItem.php
│   └── User.php (role)
```

Админка на **Filament**: ресурсы `CategoryResource`, `ProductResource`, `OrderResource`, `UserResource` — таблицы и формы создаются быстро, при необходимости кастомизируем под «автозапчасти» (артикулы, остатки, статусы заказов).

## Этапы реализации (кратко)

| Этап | Что делаем |
|------|------------|
| 1 | Laravel Breeze + роли (user/admin), миграции User (role), базовая верстка (шапка, футер, Tailwind). |
| 2 | Модели Category, Product, ProductImage; миграции; Filament: CategoryResource, ProductResource. |
| 3 | Каталог на Livewire: список категорий, ProductGrid с фильтрами, страница товара, поиск. |
| 4 | Корзина (сессия/БД): Cart, CartItem, Livewire-компоненты корзины и кнопка «В корзину». |
| 5 | Оформление заказа: Order, OrderItem, CheckoutWizard (гость/пользователь). |
| 6 | Личный кабинет: профиль, список заказов, детали заказа. |
| 7 | Админ: управление заказами (статусы), пользователями; дашборд при необходимости. |
| 8 | Доработки: уведомления (email при новом заказе), SEO (meta, slug), изображения (оптимизация/водяные знаки). |
